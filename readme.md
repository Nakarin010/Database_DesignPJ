## Database Design Project

This is a group project in Database design subject.
We select the airline business to design database and simulate real/close to real operations.

There are 5 phases in the development of project:
1. Data base Planning and Requirements Analysis.
2. Conceptual Database Design: User requiremnet and ERD.
3. Logical Database Design (ER diagram and Relation schemas & constraints).
4. Implementing phase.
5. Database Tuning.
